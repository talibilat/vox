- Interrupt latency re-validated post-tuning on real hardware (Piper voice, real speaker, audio audibly playing): stop path median 124ms, worst 140ms, within the 200ms bar for the instrumented portion; the end-to-end decomposition including frame-quantized detection lag is recorded in the protocol doc.
